# Early Project- Circle Agency

## Description
   Responsive website for a digital agency with dynamic projects and contact form.

## Folder structure
    /Early project
    │── index.html
    │── contact.html
    │── README.md
    │── /projects
        │── 1.html
    │── /css
        │── style.css

    │── /js

    │── /assets
        │── hero-section
        │── logos
        │── newsletter
        │── projects-section
        │── services-section
        │── testimonial-section


## Features
- Responsive website (mobile, tablet, desktop).
- Projects loaded from API.
- Contact form with validation.
- Hamburger menu.

## Configuration
No installation needed. Open `index.html` in a browser or use Live Server.

## Technical Specifications
- HTML / CSS / JavaScript (vanilla).
- External API: https://raw.githubusercontent.com/ironhack-jc/mid-term-api/main/projects.
- Responsive breakpoints: 768px, 1024px.
